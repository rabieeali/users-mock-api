How To Use?

  1. you should have node.js already installed on your machine
  2. go to "Backed" folder and install all the dependencies with npm install
  3. the server is by default run on port 3500
  4. in order to avoid "CORS", in the directory "config/allowedOrigins.js" change your desired frontend origin (VSCode live-server's already set!)
  5. start the server with npm run dev to run the server by "Nodemon" or npm run start by "node"
  6. provided methods are GET, POST, PUT, DELETE
  7. Checkout the API here
  8. You Need a Live-Server to run the frontend functionalities
  9. to see more great stuff follow me on github and give me a star ⭐
